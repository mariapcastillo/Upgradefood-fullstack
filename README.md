Proyecto final del Bootcamp Full Stack de Upgrade Hub.  
Aplicación web completa para la gestión de un restaurante japonés: reservas, menús, reseñas y panel administrativo, con frontend Angular y backend FastAPI conectados mediante API REST.

---

## Stack tecnológico

Frontend: Angular · HTML · CSS · JavaScript  
Backend: Python · FastAPI · REST API  
Base de datos: MySQL  
Control de versiones: Git · GitHub

---

## Funcionalidades principales

- Registro y login de usuarios
- Gestión de reservas
- Gestión de menús y platos
- Reseñas de usuarios
- Panel de administración
- Autenticación con JWT
- Protección de rutas y roles

---

## Estructura del proyecto

frontend/ → Aplicación Angular  
backend/ → API FastAPI

---

## Ejecución en local

### Backend

cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload

### Frontend

cd frontend  
npm install  
ng serve
